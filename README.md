# 🌿 App de Plantas Medicinais e Saúde Alternativa

Aplicativo desenvolvido como Trabalho de Conclusão de Curso (TCC), com foco em saúde alternativa e no acesso à informação sobre plantas medicinais e profissionais da área.

O app tem como objetivo reunir, de forma simples e acessível, dados relevantes sobre diversas plantas medicinais e conectar usuários a profissionais especializados, promovendo o uso consciente e informativo de recursos naturais.

---

## 📱 Sobre o projeto

Este aplicativo permite que os usuários consultem informações detalhadas sobre plantas medicinais, incluindo seus usos terapêuticos, formas de cultivo e evidências científicas.

Além disso, o sistema conta com funcionalidades inteligentes, como a identificação de plantas por imagem e a busca por profissionais da área de saúde alternativa próximos à localização do usuário.

---

## 🚀 Funcionalidades

- 📷 Identificação de plantas por imagem  
- 🌱 Listagem de plantas medicinais  
- 🔎 Visualização de informações detalhadas:
  - Nome científico  
  - Usos medicinais  
  - Onde encontrar  
  - Como cultivar  
  - Pesquisas relacionadas  
- ⭐ Sistema de favoritos  
- 🔍 Busca e navegação intuitiva  
- 📍 Busca por profissionais de saúde alternativa próximos:
  - Médicos homeopáticos  
  - Localização baseada em proximidade  

---

## 🛠️ Tecnologias utilizadas

- React Native  
- TypeScript  
- [API de reconhecimento de imagem]  
- [API de geolocalização]  
- [Banco de dados SQL]

---

## 🎯 Objetivo

Facilitar o acesso a informações confiáveis sobre plantas medicinais e aproximar os usuários de profissionais da área de saúde alternativa, incentivando práticas mais naturais, seguras e conscientes.

---

## 🎨 Interface

O aplicativo foi desenvolvido com foco em usabilidade e ergonomia, apresentando uma interface simples, organizada e intuitiva, proporcionando uma boa experiência ao usuário.

---

## 📚 Contexto

Este projeto foi desenvolvido como requisito para conclusão de curso, integrando conhecimentos de desenvolvimento mobile, design de interface e pesquisa na área de saúde alternativa.
